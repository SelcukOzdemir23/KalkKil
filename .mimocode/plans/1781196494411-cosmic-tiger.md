# KalkKıl — Tasarım İyileştirme Planı

## Mevcut Tasarım Analizi

### Kıble Bulucu Sorunları
1. **Pusula çok minimal** — Sadece 6px genişliğinde altın renkli bir çizgi. Kardinal yönler (K/D/G/B) yok, açı göstergeleri yok. Kullanıcı neye bakacağını bilemiyor.
2. **Harita baskın, pusula arka planda** — Tam ekran WebView haritası pusulayı gölgeliyor. Pusula haritanın ÜZERİNDE ama pointerEvents:'none' olduğu için dokunulmuyor — bu doğru ama görsel olarak pusula haritaya karışıyor.
3. **Alt panel haritanın altını kaplıyor** — Kıble yönü bilgisi paneli, tab bar ile çakışabilir.
4. **Hata durumları pasif** — Konum yokken veya sensör yokken sadece mesaj gösteriyor, kullanıcıyı harekete geçirmiyor.

### Genel Uygulama Sorunları
1. **CountdownTimer çok yüksek** — 42px prayer name + 28px countdown = ~180px yükseklik. ScrollView'da çok yer kaplıyor.
2. **PrayerCard'lar monoton** — Tüm kartlar aynı genişlikte, sadece renk/opaklık farkı. Her vakit için ikon yok.
3. **HomeScreen arka plan dekoru basit** — Sadece sağ üstte tek daire.
4. **Tab bar 68px** — biraz kalın, 60-64px daha optimal olur.

## Önerilen Değişiklikler

### 1. Kıble Bulucu — Tamamen Yeniden Tasarım

**Mevcut:** Harita tam ekran, pusula üzerinde küçük çizgi
**Yeni:** İki katmanlı tasarım — üstte büyük pusula dairesi, altta mini harita

#### Yeni Kıble Ekranı:
```
┌─────────────────────────────┐
│     [Başlık: Kıble]        │
│                             │
│    ┌───────────────────┐    │
│    │   N               │    │
│    │      •            │    │  ← Büyük pusula dairesi (200px)
│    │  B─ ┼ ─D          │    │     Kardinal yönler: N/S/E/W
│    │      ↑            │    │     Kıble ibresi (altın)
│    │   S               │    │     Kullanıcı yönü (yeşil nokta)
│    └───────────────────┘    │
│                             │
│  "247° GKB"                 │  ← Kıble yönü açısı
│  "Telefonu Kâbe'ye çevirin" │
│                             │
│  ┌───────────────────────┐  │
│  │  Mini Harita (150px)  │  │  ← Leaflet haritası (küçük)
│  │  Konum ─── 🕋         │  │
│  └───────────────────────┘  │
│                             │
│  ┌───────────────────────┐  │
│  │  🟢 Kıbleye dönük!   │  │  ← Durum kartı
│  │  Namazınızı kılabilir │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

**Değişiklikler:**
- `QiblaScreen.tsx` — Harita ile pusula yer değiştir. Pusula öne çıkıyor, harita arka planda kalıyor.
- Pusula dairesi: 220px çap, kardinal yön harfleri (K/D/G/B veya N/S/E/W), açı çizgileri
- Kıble ibresi: Kalın (8px), altın rengi, ucunda ok işareti
- Kullanıcı yönü: Yeşil nokta pusula merkezinde
- Mini harita: 150px yükseklikte, sadece referans için
- Hata durumunda "Ayarlara Git" butonu

### 2. CountdownTimer — Kompakt Tasarım

**Mevcut:** ~180px yükseklik, 42px prayer name
**Yeni:** ~120px yükseklik, daha kompakt ama etkili

```
┌─────────────────────────────┐
│  SIRADAKİ VAKİT             │
│  İkindi  16:45              │  ← 24px prayer, 16px time
│  ─────────────────          │
│  02:34:15                   │  ← 22px countdown, daha küçük
└─────────────────────────────┘
```

**Değişiklikler:**
- `CountdownTimer.tsx` — Fontları küçült (42→28, 28→22, 20→16)
- Padding azalt (24→16)
- Toplam yükseklik ~120px'e düşsün

### 3. PrayerCard — Korunuyor
PrayerCard mevcut haliyle kalıyor (nokta göstergesi + isim + saat). Sadece geri sayım ve kıble ekranı yeniden tasarlanacak.

### 4. HomeScreen — Arka Plan ve Akış

**Değişiklikler:**
- Arka plan dekorunu büyüt ve ortala (sağ üst tek daire yerine, üstte büyük yarım daire gradient)
- Konum etiketini daha belirgin yap
- AlertModal'ı iyileştir (şimdiki zaten iyi)

### 5. Tab Bar — İnceleştirme

**Değişiklikler:**
- `App.tsx` — Tab bar yüksekliğini 68→60px düşür
- borderRadius'u 24→20px yap (daha doğal görünüm)
